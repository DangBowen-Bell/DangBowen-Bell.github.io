3D Human Reconstruction from RGB-D Data with Occlusions

Bachelor Thesis & Master Research Project

Mar 2021 - Present

Project Description: The project aims to reconstruct the 3D human body from an RGB-D image and the scene mesh. The reconstructed human pose should be accurate and plausible, with minimal penetration and necessary contact with the scene.

Personal Work: Using deep learning methods to extract the interaction features between the human body and the scene. These features are utilized as a constraint to assist the reconstruction process. Experiments were conducted on multiple datasets containing occlusion, and the results showed a significant improvement compared to methods that did not consider interaction.

Server Log Anomaly Detection System

National College Student Innovation and Entrepreneurship Training Program

Dec 2018 - Jun 2020

Project Description: The project aims to use machine learning algorithms to detect anomalies in system logs, replacing traditional rule-based matching methods, reducing labor costs, and improving efficiency.

Personal Work: Using machine learning algorithms to detect anomaly information after extracting different types of features from the raw system logs. The results from different algorithms are evaluated, and a web-based platform is built to visualize the detection process and display the detected anomalies. The project received an excellent rating in the final project defense. The solution ranked 3rd out of 269 participants in the Huawei Log Anomaly Detection competition.
